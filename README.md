# Online bicycle shop

### General info
Online store with bicycles.

I plan to extend the application with new functionalities and Spring Security.

### Technologies
- Java
- Spring
- Spring MVC
- Spring Data
- Hibernate
- JSP

### Features
- registration and login
- products review
- product details
- adding to the cart
- finalizing the purchase
For administrator:
- adding, editing and deleting products
